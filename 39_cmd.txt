linux常用基本命令：
	命令名 [选项参数] [操作源对象] [操作目标对象]
目录操作命令：
	ls	列出当前目录下的内容
		目录：文件夹
		-a	查看所有文件，包括隐藏文件
			linux下以.开头的文件是隐藏文件
		-l	查看文件的详细信息
		-S	按文件大小进行排序
		-t	按时间排序
		-r	按原先的排序进行逆序显示
	pwd	显示当前所在路径
		显示的是当前所在路径的绝对路径
	mkdir	创建目录
		-p	从外往内递归创建目录
	rmdir	删除空目录
		-p	从内往外判断是否为空，递归删除空目录
	cd	改变当前所在路径
		~	当前用户的家目录（每个用户都不同）
		-	上一次所在路径
	rm	删除文件
		-r	递归删除目录中的所有文件以及目录本身
		-f	忽略提示信息
	cp	拷贝文件到指定目录下
		-r	递归拷贝目录
		-f	忽略提示信息
	mv	将一个文件/目录移动到指定的目录下
		（对文件/目录改名）
		-i	增加提示信息
文件操作命令:
	touch 	若文件不存在则创建，存在则刷新时间属性
		-d	"2004-02-29 16:21:42"
		-t	[[CC]YY]MMDDhhmm[.ss]
		刷新的是最后一次访问和修改时间
	一个文件的时间属性有：
		最后一次访问时间
		最后一次修改时间
		之后一次状态改变时间
	cat	打印文件内容到显示终端
		-n	每一行带有行号
	more	对文件内容进行分页显示
		空格	向下翻页
		q	退出
	less	对文件内容进行分页显示
		f 空格 pgdown	向下翻页
		b pgup		向上翻页
		j k  上下箭头	上下按行移动
		/string		向下匹配string字符串
		？string	向上匹配string字符串
			n	继续匹配下一处位置
		q	退出
	head	显示文件的前n行内容，默认是10行
		-n 2	用于指定显示文件前2行
	tail	显示文件的末尾n行内容，默认是10行
		-n 2 	用于指定显示文件末尾2行
		-f	动态刷新文件末尾数据
		ctrl+c	中断当前操作
压缩解压缩操作：
	创建一个100M大小的文件
		dd if=/dev/zero of=./hello.txt bs=100M count=1
	压缩/解压缩/打包
	zip/unzip
		zip hello.zip hello.txt	
		将hello.txt文件压缩生成一个叫hello.zip的压缩包
	gzip/gunzip
	bzip2/bunzip2
	tar	打包命令
		-c	打包
		-x	解包
		-v	显示详细的打包信息
		-z	打包/解包同时进行gzip格式压缩/解压缩
		-j	打包/解包同时进行bzip2格式压缩/解压缩
		-f	用于指定压缩包
其他命令 ：
	ifconfig	查看主机IP地址信息
	date	查看系统时间
	man	用于查看命令/函数手册
	echo	打印内容到显示终端上
	su	切换当前用户（但是不切换当前所在路径）
		上一个用在什么路径下，那么切换后还在这个路径下
	linux下一切皆文件，linux下并不以文件后缀名区分文件类型
	linux下的目录结构是一个树型结构，最上层的目录是根目录/

	绝对路径：以根目录为起始查找路径的路径叫绝对路径
	相对路径：以当前所在路径为起始查找路径叫相对路径
	
linux操作系统是一个多用户，多任务操作系统
